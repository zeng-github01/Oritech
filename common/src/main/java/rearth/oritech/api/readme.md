The folder in this directory contain cross-platform APIs for item, energy and fluid transfer, used by both the pipes and machines in Oritech.
There is also a cross-platform attachment api for attaching data to the player.

Feel free to take and use any of the code here in your project. It is designed somewhat generic, but a few things are tailored toward the use-cases of oritech,
so don't expect a perfectly polished cross-platform lib here.
