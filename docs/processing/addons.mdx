To upgrade machines in Oritech, addons are used. They are blocks that need to be attached to the machine itself or a connected machine extender.
Addons can perform a variety of functions, such as increasing speed, improving energy efficiency, providing access to specific inventory slots, and more.

Machines can only accept addons at specific positions. To view these, either check the "addons" UI page or look for these markers on the machine:

<center>
<ModAsset location="oritech:area/addon_marker" width={300} />
</center>

Addons are activated when the machine is right-clicked. To indicate activation, the pink parts of an addon turn blue when in use.

<Callout variant="warning">
    Always make sure you open the machines' UI after adding addons to ensure that they are connected. The addons will change color from pink to blue.
</Callout>

To extend the available number of addon slots, you can use machine extenders. These specific addons don't directly influence the machine but allow
addons to be placed on them, which then count towards the machine they are connected to.

The maximum layers of machine extenders you can use depends on the machine quality. If you have a machine with a core quality of 1,
you can use 1 addon in total. The total number of machine extenders you can use in a machine is equal to the machine quality.

## Burst Addon

<center>
<ModAsset location="oritech:machine_burst_addon" width={256} />
</center>

The burst addon is a special addon that can be added to any processing machine. It allows the machine to operate in much faster bursts (8 times the normal speed). This
speed can be combined with other addons to further increase the speed. However, it can only operate faster in bursts. If it operates in burst mode for too long, it overheats
and works much slower. It needs to stop processing for a while before it can burst again.

Each burst addon increases the maximum burst time by 12 seconds. Multiple burst addons can be used to increase the maximum burst time. However, for machines that should run
permanently, burst addons are not recommended.
