[プレイサーブロック](../interaction/block_placer)、[デストロイヤーブロック](../interaction/block_destroyer)、および[ファーティライザーブロック](../interaction/fertilizer)はすべて、マシンフレームで構築されたガントリー上で動作します。マシンフレームとは、機械が動作する作業領域を指定するもので、長方形で内部が空の状態である必要があります。

機械は常にフレームの**下側**にあるブロックをターゲットとします。

<Callout variant="warning">
    フレームの角部分には機械を設置できませんのでご注意ください。
</Callout>

同じマシンフレーム上で複数の機械を同時に稼働させることが可能です。この場合、単に複数の機械をフレーム上に配置するだけで構いません。機械はフレーム領域内のすべてのブロックに対して順次処理を行います。

マシンフレーム上で動作するすべての機械は、ほとんどのアドオンを使用することができます。

スピードアドオンを使用すると、フレーム内で動作する機械の移動速度と作業速度の両方が向上します。
