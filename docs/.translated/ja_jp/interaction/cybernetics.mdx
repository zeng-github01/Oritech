サイバネティックオーグメントセンターは、身体にサイバネティックオーグメントを装着できる施設です。
これらのオーグメントには、筋力や速度の向上、防御力の強化など、さまざまなメリットがあります。
また、暗闇でも視認できる能力や、自動的に栄養補給が行われる機能など、新たな能力を付与することも可能です。

<Callout variant="info">
    オーグメントはプレイヤーに永続的に適用されるアップグレードであり、維持のためにエネルギーやリソースを消費することはありません。
</Callout>

### 研究要件
すべてのオーグメントは、まずサイバネティックオーグメントセンターで研究する必要があります：
1. 組み立てられた機械の中に入って、「研究＆適用」GUIを開きます
2. 各オーグメントには独自のエネルギー消費量とリソースコストがあります（これらはセンターのインベントリまたはプレイヤーのインベントリに保存されます）
3. 機械のインベントリにアクセスするには、機械を右クリックするか、GUIの左下にあるボタンを使用します
4. 多くの研究には、技術ツリーに示されているように前提条件となるオーグメントが必要です

### 研究ステーション
- オーグメントセンターには3つのマルチブロック研究スロットがあります（アドオンスロットと同様の仕組みです）
- 各ステーションの要件：
  - 3つのマシンコアが必要
  - 1つの研究を同時に処理可能
  - 同じ種類のステーションを複数設置すれば、並行研究が可能になります

### 適用プロセス
1. 研究が完了したオーグメントは装着可能になります
2. 適用には追加のリソースが必要です
3. センターには研究したすべての設計図が保存されます（機械ごとに保存されるため、機械が破壊されるとデータは失われます）
4. 既存のプレイヤー用オーグメントは、UIのインポートボタン（左下）を使用してスキャンできます

### オーグメントの管理
- すべてのオーグメントは永久的に装着されます（維持やエネルギー消費は不要です）
- 一部のオーグメントは切り替え可能ですが、常に有効化されているタイプもあります
- オーグメントはいつでも取り外し可能ですが、その装着に使用したリソースは失われます

### 切り替え操作
1. オーグメントのホットキーを押し続けます（デフォルトは[G]キー）
2. 放射状UIで目的のオーグメントにカーソルを合わせます
3. キーを離すと切り替えが完了します

### modpack開発者向け情報：
オーグメントは完全にカスタマイズ可能でデータ駆動型です。新しいオーグメントはデータパックを使用して作成できます。オーグメントには3つのタイプがあります：
プレイヤーの属性を変更する「モディファイアオーグメント」（速度やリーチなどの属性に影響します）、ポーションと同様の効果を付与する「効果属性オーグメント」、そして鉱石視認機能やポータルオーグメントなどの「カスタムオーグメント」です。これらのカスタムオーグメントはデータパックで参照可能ですが、そのロジックは完全なJava modとして実装する必要があります。

完全に新しいオーグメントはデータパックで作成可能で、既存のオーグメントも変更できます。オーグメントの定義例については、以下のファイルをご参照ください：https://github.com/Rearth/Oritech/blob/1.21/common/src/main/generated/data/oritech/recipe/augment/attackdamage.json

新しいオーグメントを追加する際には、翻訳エントリとテクスチャも必要になることをご注意ください。テクスチャは「oritech/textures/gui/augment」ディレクトリに配置してください。
