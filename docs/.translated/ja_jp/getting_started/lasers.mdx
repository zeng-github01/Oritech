さらに先に進むためには、まず[フラクサイト](../resources/fluxite)が必要です。これを手に入れるには、[エンダリックレーザー](../interaction/enderic_laser)を使ってアメジストの結晶を過負荷状態にします。

エンダリックレーザーの製作は、これまでの工程を発展させたものです。合金の組み合わせ、組み立てた部品、そして[エンダリック化合物](../resources/enderic_compound)やカーボンファイバーの糸などの材料が必要になります。
これらの材料は、[遠心分離機](../processing/centrifuge)と[粉砕機](../processing/pulverizer)を使って製造します。

新しく製作したエンダリックレーザーと[ターゲットデザイナー](../tools/target_designator)、そして十分な電力があれば、フラクサイトの生成が可能になります。

フラクサイトを入手するには、エンダリックレーザーでアメジストの結晶を採掘します。ターゲットデザイナーを使ってレーザーをアメジストの結晶に向けます。
成長途中のアメジストは破壊されることなく、むしろエネルギーが与えられ、成長が促進されます。レーザーは自動的に進行方向にあるアメジストを標的にします。
