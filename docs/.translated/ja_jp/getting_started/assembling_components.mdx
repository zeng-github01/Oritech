[組み立て機](../processing/assembler)を構築してみてください。

組み立て機は、ワイヤーやコイル、モーターなどの部品を、あなた自身で組み立てるよりも効率的に製造できます。また、クラフトでは製造できない部品の製造も可能です。

[遠心分離機](../processing/centrifuge)を構築してみてください。

遠心分離機は、特定の資源を処理し、さらなる工程で必要となる部品を製造するために必要となります。

これらの機械はいずれも、レッドストーン・フラックス（RF/FE）という形態のエネルギーを必要とします。また、アドオンを使用することで機能をアップグレードすることが可能です。
