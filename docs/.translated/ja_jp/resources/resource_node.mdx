資源ノードは世界の至る所に存在し、岩盤のすぐ上の層に密集して生成されます。
これらのノードにはオーバーワールドで採掘可能なあらゆる種類の鉱石が含まれており、通常は少量の[原油](../resources/crude_oil)の膜に覆われています。

資源ノードの採掘には、[岩盤ドリル](../interaction/deepdrill)を使用する必要があります。

地上では、[鉱石の塊](../resources/ore_boulder)の存在によって、資源ノードの群生地の位置を特定できます。
この鉱石の岩塊は、群生地に含まれる鉱石の種類を示す指標にもなります。
