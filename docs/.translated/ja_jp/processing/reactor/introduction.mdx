## 概要

**Oritech社製原子炉**は、大量の電力を生成する優れた装置です。
ウランまたはプルトニウムの燃料棒を燃料として使用し、燃料棒の本数と燃料棒周辺の中性子反射板の数に応じてRFエネルギーを発生させます。
ただし、十分な冷却を維持することが非常に重要です。

<center>
<ModAsset location="oritech:area/reactor" width={1000} />
</center>

原子炉内で燃料を燃焼させると熱が発生します。原子炉の熱を管理するためには、以下の複数の要素が必要です：
- **熱パイプ**：燃料棒から熱を吸収し、パイプ内を移動させて放熱します。
- **原子炉用排気口**：最も高温になっている隣接部品（燃料棒や熱パイプなど）から一部の熱を取り出し、原子炉外部に排出します。
- **熱吸収体**：すべての隣接部品から一定量の熱を吸収しますが、冷却材アイテムを継続的に供給する必要があります。

原子炉にはある程度の熱が蓄積されます。理想的には、適切に冷却された原子炉からは熱が漏れ出ることはありません。ただし、熱排気口などの部品は、高温環境下でより効率的に機能します。過剰な熱が発生し、適切に除去されない場合、燃料棒が過熱し、核融解を引き起こす可能性があります。2000°Cを超える温度が長時間続くと、核融解が誘発されます。原子炉がこの危険な温度に達する前に、警告サイレンと視覚的なインジケーターが作動します。


<Callout variant="info">
    Oritech社の見習い科学者の方々で、新しい原子炉設計やバリエーションを安全に探求したい方には、「安全原子炉」という設定オプションが用意されています。この設定を有効にすると、原子炉は過熱時に爆発するのではなく、単に冷却期間に入るようになります。
</Callout>

原子炉は長方形の壁で構成する必要があり、壁の端部分は**原子炉用ウォール**ブロックで構築します。原子炉の最大サイズは各方向64ブロックです。
**原子炉コントロールパネル**は壁のどこかに設置する必要があります。**原子炉用エネルギーポート**は側面壁に設置することも可能です。
その他の非ポート部品はすべて原子炉の内側に配置してください。原子炉の全高は任意に設定できますが、内部構造は垂直方向で同一である必要があります。つまり、**2D空間で原子炉を設計した後、それを任意の高さに垂直方向に拡大することが可能**です。内部高さ1で動作する原子炉は、内部高さ10の場合も同様に動作しますが、エネルギー生産量や冷却材・燃料消費量はそれに応じて調整されます。
