アトミックフォージは、高度なコンピュータチップの製造、深層岩の強化、さらには鉱石の複製まで可能な強力な機械装置です。

<Callout variant="warning">
    エネルギーパイプは直接アトミックフォージに接続することはできません。[エンダリックレーザー](../interaction/enderic_laser)で電力を供給する必要があります。
</Callout>
