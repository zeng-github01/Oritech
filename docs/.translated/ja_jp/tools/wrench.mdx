### 説明
パイプレンチはパイプの接続や切断に使用する工具です。パイプの各方向の接続状態の編集や、パイプを簡単に除去することができます。

### 使用方法
#### 配管接続の切り替え
すべてのパイプは、それぞれの方向に対しての接続状態を切り替えることができます。パイプを分断するには、
すでにつながっている部分を右クリックします。接続を有効にするには、パイプの中心部分を右クリックし、
接続を有効にしたい方向に対応する側面を右クリックします。


#### 配管の切断
配管を切断するには、パイプレンチを持った状態でシフトキーを押しながら配管を右クリックします。配管は即座にアイテムとしてドロップします。
また、パイプレンチをピッケルのように使用して配管を切断することも可能です。パイプレンチは配管切断時の採掘速度が通常のピッケルよりも速くなっています。
