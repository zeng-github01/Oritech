エンダリックレールガンは、高速で飛翔体を発射する強力な遠距離武器です。
敵に対して大きなダメージを与えることができ、遠距離からブロックを破壊することも可能です。
この武器はRF（エネルギー）で動作し、使用前に充電が必要です。

左マウスボタン（攻撃/採掘操作）を押し続けると、連続ビームが発射され、進路上のあらゆるエンティティにダメージを与えます。また、直線上の最初のブロックを採掘します。
エネルギーを蓄えられるブロックに対しては、ブロックを破壊する代わりにエネルギーを送信します。また、エンダリックレーザーと同様に採掘したブロックを変換する機能も備えています（例：フラクサイトの採掘）。

採掘モード/ブロック破壊機能は、アイテムを所持した状態でShiftキー+右クリックすることで切り替え可能です。

右マウスボタンをクリックすると、強力な飛翔体が発射され、標的地点で爆発します。これにより周辺のブロックやエンティティが破壊されます。
