プロメテウム製ピッケルは、耐久性に優れ、壊れない設計で、さらにエンチャント可能な高度な採掘ツールです。

このピッケルには2つのモードがあります：**シルクタッチ**モードと**範囲採掘**モード（3x3範囲採掘）です。モードを切り替えるには、シフトキーを押しながら右クリックしてください。

<ModAsset location="oritech:promethium_axe" />

プロメテウム製バトルアックスもまた、耐久性に優れ、壊れない設計で、さらにエンチャント可能な高度な切断ツールです。
