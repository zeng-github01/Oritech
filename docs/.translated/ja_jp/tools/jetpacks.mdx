空を飛ぶ時が来ました。Oritech社では様々なタイプのジェットパックを提供しています。すべてのモデルで、エネルギーまたはターボ燃料のいずれかを使用して飛行可能です。
エネルギータンクは、任意のエネルギーストレージデバイスまたは充電器を使用して補充します。内臓のターボ燃料タンクを充填するには、専用の充電器ブロックが必要です。

ターボ燃料を供給すると、ジェットパックの飛行速度が大幅に向上します。Exoジェットパックは標準モデルよりも高速で、上昇速度が速くなるだけでなく、
前進速度も向上し、上昇する必要なくより速く移動できるようになります。

ジェットパックはオン/オフの切り替えができません。ジャンプキーを押し続けている間は常に作動状態になります。

基本型ジェットパックとExoジェットパックには、それぞれエリトラ版が用意されています。この場合、ジェットパックはユーザーがエリトラで飛行している時のみ作動し、花火を使用した場合とは異なり、
上昇方向への補助ブーストを提供します。
