Exoアーマーは、冒険をサポートするために設計された専用装備です。各パーツにはそれぞれ独自の効果があります：

- **ヘルメット:** 暗視機能を提供します。
- **チェストプレート:** インベントリ内のアイテムを充電でき、鉱山での滞在時間を延長できます。
- **レッグガード:** 移動速度を向上させます。
- **ブーツ:** 落下ダメージに対する耐性を付与します。
